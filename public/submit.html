Atlas Contributor Portal

Submit new mathematical definitions, theorems, and proofs to the graph

Required Format

Step 1: Write your content in Typst markdown
Step 2: Add metadata comments at the top
Step 3: Upload the file and we'll validate it

Your submission must include these three metadata lines at the beginning:

  // id: thm-my-theorem
  // type: theorem
  // deps: [def-group, ax-associativity]
  ---

  Let $G$ be a group. Then...

Metadata Reference

  1. Id: Unique identifier (e.g., thm-bolzano-weierstrass)
  2. Type: One of: theorem, lemma, definition, axiom, intuition, proof
  3. Deps: Array of IDs this node depends on (can be empty: [])
Valid ID formats:
thm-fermat-last, def-prime-number, ax-choice, lem-triangle-inequality
